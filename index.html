Welcome to EthSlurp

EthSlurp allows you to 'slurp' the Ethereum blockchain extracting transactions from any Ethereum address, including smart contracts. For example, 

 ethslurp 0xbb9bc244d798123fde783fcc1c72d3bb8c189413

returns all transactions from "The DAO." If you run the above command (don't it takes forever) you will get JSON data, i.e, computer gobbeldy-gook.

This is where EthSlurp comes in. Using the formatting (-f) option makes the output human readable. 

 ethslurp -f 0x713B73c3994442b533e6A083eC968e40606810Ec

It's even better than that. The formatting is fully customizable using a feature called display strings. Choose which fields to include, which fields to ignore, and exactly what format the output takes.

For example, by adding a formatting options to the -f command, you can create a tab delimited text file, a comma seperated .csv file that will open directly in MS Excel, or an even rudimentary HTML. 

 ethslurp -f:txt 0x713B73c3994442b533e6A083eC968e40606810Ec
 ethslurp -f:csv 0x713B73c3994442b533e6A083eC968e40606810Ec
 ethslurp -f:html 0x713B73c3994442b533e6A083eC968e40606810Ec

There are many other options detailed fully in our documentation.

Why would I need to do that?

Let's say you've deployed a smart contract. How do you know what's going on with it? With EthSlurp you can create the information you need to fully account for all the incoming and outgoing ether in your contact. You can even filter the transactions by date. Wouldn't it be nice to finally know on a monthly or weekly basis if your contract is making money?

Building the software

Currently (pre-beta), we do not provide binaries (mostly because we don't know how). You must download the source code and compile it yourself--but don't worry--it's extremely easy. Information on building the software is provided on the documentation page.

We also do not know how to make a full-fledged package. There may be a few dependencies which may not resolve during the build. Please contact us if that happens, and we will respond quickly. Email us with questions: info@ethslurp.com
